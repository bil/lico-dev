Linux Comodular Realtime Interactive Computation Engine (LiCoRICE) is a tool that allows for prototyping and running realtime systems on conventional POSIX hardware. LiCoRICE allows users to write high-level Python code blocks that are translated to C and compiled to run in a model-based design infrastructure. This maintains the flexibility and library support of Python while enabling realtime timing guarantees.
